Manipulando o DOM

Document Object Model - Criado pelo W3C, O DOM é uma multi-plataforma que representa como as marcações em HTML, XHTML e XML são organizadas e lidas pelo navegador que você usa. Uma vez indexadas, estas marcações se transformam em elementos de uma árvore que você pode manipular via API – que é o que fazemos quando usamos programas ou scripts para alterar funcionalidades de uma página: conteúdo, estrutura ou folha de estilo.

  • fonte: https://tableless.com.br/tenha-o-dom/
// pegar um elemento
  const app = document.getElementById('app');
  // criar um elemento
  const spanHello = document.createElement('span');

  // adicionar atributos
  spanHello.className = 'my-span-class';
  spanHello.id = 'my-span-id';
  // criar um nó de texto
  const text = document.createTextNode('Hello');
  // adicionar filhos
  spanHello.appendChild(text);
  app.appendChild(span);

  const myLink = document.createElement('a');
  myLink.setAttribute('href', 'http://google.com');

  app.appendChild(myLink);

  // remover filho
  app.removeChild(myLink);

Event Listeners

addEventListener() registra uma única espera de evento em um único alvo. O alvo do evento pode ser um único elemento em um documento, o documento em si, uma janela, ou um XMLHttpRequest.

Para registrar mais de uma espera de evento como alvo, chame addEventListener() para o mesmo alvo mas com diferentes tipos de evento ou captura de parâmetros.

fonte: https://developer.mozilla.org/pt-BR/docs/Web/API/Element/addEventListener

const input = document.getElementById('myInput');

  input.addEventListener('keyup', event => {
    console.log(event.target);
    console.log(event.target.value);
  });

  const button = document.querySelector('#myButton');

  button.addEventListener('click', event => {
    console.log(event.target);
    console.log(event.target.value);
  });

Foreach, Map, Filter e Reduce

Essas funções são extremamente usadas, portando dê uma atenção.

Recomendo de fato entrar no blog e dar uma boa olhada e fazendo seus próprios testes. Reduce é uma das funções mais ricas, mas também um pouco mais difícil de entender, mas não tema!

const data = [
  { name: 'Galadriel', race: 'Elves', weapons: [ 'Elven Magic', 'Nenya' ] },
  { name: 'Legolas', race: 'Elves', weapons: [ 'Bow', 'Knife' ] },
  { name: 'Gandalf', race: 'Maiar', weapons: [ 'Glamdring', 'Wizard Staff', 'Sword' ] },
  { name: 'Radagast', race: 'Maiar', weapons: [ 'Powers of the Maiar', 'Wizard Staff' ] },
  { name: 'Aragorn', race: 'Men', weapons: [ 'Anduril', 'Sword' ] },
  { name: 'Sauron', race: 'Ainur', weapons: [ 'One Ring', 'Sword', 'Powers of the Maiar' ] },
  { name: 'Faramir', race: 'Men', weapons: [ 'Bow', 'Sword' ] }
];

// mostrar os nomes
data.forEach(item => {
  console.log(item.name);
});

const getNames = data.map(item => item.name);
console.log(getNames);

const filterOnlyTheElves = data.filter(item => item.race === 'Elves');
console.log(filterOnlyTheElves);

const countTheWeapons = data.reduce((acc, item) => {
  acc += item.weapons.length;
}, 0);

console.log(countTheWeapons);
